Roderick, Lionel, and Yusuke talked about possible game features.


Basic story:
Space creatures start invading the earth and they have destroyed
the entire vegetation of the earth except for one single tree. The only hope is to heal the tree and fight the enemies to revocer the beautiful planet.

Game Characters:

Fighters: the avatar the player controls. He has the ability to fight against the enemies and to heal the plant.

Enemies: So far, enemies include butterflies, caterpillars, virus, and so on with different feaures.

Weapons: Spears, hoes, shovels, sprays, etc.

As game goes on, the tree keeps growing.

This game can involve multiple players with different abilities.
While some are specialized in fighting, others can contribute by healing the tree faster.
