GAME

Game Discription

We are currently working on a third-person-perpective, singleplayer tower-defence game.

One day, aliens began attacking on the earth. While they won't attack human beings, their purpose is to destroy the vegetation of the planet.
Luckly, you, as a brave soldier, have the ability to build defensive turrets to attack the enemies.
Your goal is to place the turrets wisely, to make sure the mother plant will survive the evil invaders.

Details

Rules

Destroy the enemies before they kill the mother plant.

Each time the player kills an alien, the player earns points.

Use the points to set defensive turrets.

Obstacles

Enemies are repeatedly generated and attack the mother plant.

Player needs to earn points before building turrets.

Player can build turrets only in a designated area.

The consecutive wave will contain enemies that have stronger attack point and more HP.

Game Level

Enemies are spawned within a certain amount of time called "wave."

Between waves, there is a time period called "break."

Every consequitive level consists of stronger enemies; Faster, more HP, more attack point, etc.

There are three kinds of enemies and multiple kinds of enemies can be spawned in a single wave.

Game Surprise: there are boss enemies with low speed, but very high HP.

Rewards: as the player kills the enemy, s/he is awarded with points, which can be used to build more defensive turrets.

Game Clear

Help the tree survive until the last wave finishes.
